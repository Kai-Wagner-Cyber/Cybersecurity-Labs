# Cybersecurity Labs

This repository documents hands-on cybersecurity labs conducted in isolated virtual environments.
The focus is on understanding how security controls affect attacker visibility, detection, and risk.

## Goals
- Build practical attacker and defender perspectives
- Understand real-world security behavior, not just tools
- Develop foundational skills for cloud and security engineering roles

## Lab Categories
- Network Reconnaissance
- Endpoint Security
- Identity & Access Management
- Cloud Security (planned)

## Environment
- VirtualBox
- Kali Linux (attacker)
- Windows (victim)
- Isolated Host-Only networking

All labs are performed ethically in controlled lab environments.
